In this project, the K-Means Clustering algorithm was used and the K value was determined by the Elbow method. Its usage purpose is very wide, it can be used for all activities where you can group. Let's imagine a shopping mall management with this algorithm, which falls into the unsupervised learning group, and it records the customers' information thanks to the shopping mall cards. In this project, I made grouping for customers.

More about K-Means Clustering: https://en.wikipedia.org/wiki/K-means_clustering

The data set includes customers' ID, annual income and spending scores given to them by the Store.
