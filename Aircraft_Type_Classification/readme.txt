The aim of the project is to analyze military aircraft models in satellite photographs using Deep Learning.
It is detection and classification. 
Our data set consists of 5 classes.
VGG-16 pre-trained complex deep learning model was used. 
(Fine tuning process has been tried.)

More info about VGG-16: https://www.geeksforgeeks.org/vgg-16-cnn-model/

The data set was obtained from MTARSI.
