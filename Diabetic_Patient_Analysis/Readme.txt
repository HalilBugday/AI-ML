The purpose of this project is to make predictions based on the analysis results of a new woman that the model decisions she is a patient or not. In simpler words, it aims to predict diabetes using the patient analysis data set.

K-Nearest Neighbors Model (KNN) was used in the project.

See About Algorithm: https://en.wikipedia.org/wiki/K-nearest_neighbors_algorithm

The data set consists of analysis results of 768 Indian women.
