In the project, emotion analysis software was developed using the NLP (Natural Language Processing) concept. The data set was obtained from kaggle.com.

More about NLP: https://en.wikipedia.org/wiki/Natural_language_processing

The data set consists of comments made by users on IMDb. Our goal is to develop an artificial intelligence model that can distinguish positive and negative comments.
